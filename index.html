CSS Positioning Cheatsheet

Position Properties & How To Use Them
Example Explanation & Definition
Static

position: static;
top: 15px;

Static

This is the default property for html/css elements so you will notice no change when you hover. This property keeps the targeted element in the flow of the document and it behaves in a normal way.
Absolute

position: absolute;
top: 40px;
left: 30px;

Absolute

This property positions the targeted element in reference to the parent element. An offset property will be offsetting the element from the parent element. This property also removes the element from the flow of the document, meaning no place is saved for it once offset. It can be positioned overlapping, above, or underneath other elements.
Relative

position: relative;
top: 40px;
left: 40px

Relative

This property positions the targeted element relative to its original place. Offset properties are positioning it relative to its default location and stays in the noraml flow of the document. Meaning if it is moved from its original place the space it occupied will remain and will not be taken by other elements.
Fixed

position: fixed;
top: 90px;

Fixed

This property removes the targeted element from the flow of the document meaning no space is saved for it. The element is positioned in reference to the 'containing block' meaning the screen. Meaning that when you scroll it stays at the same place on the screen. This can be helpful for elements from want to stay visible throughout a document such as a navigation bar, breadcrumb map, or list of chapters/sections.
Sticky

position: sticky;
top: 120px;

Sticky

This property positions the target element in reference to its closest block level ancestor and is offset from this ancestor. The element stays in the normal flow of the document, meaning its deafult place is saved. But it scrolls to remain in the same position on the screen. This has many of the capabilities of the fixed position. It depends on whether you want it positioned in reference to ancestor or in reference to the body

Offset Properties

top: 0px;

The top offset moves an element down. Because it is offsetting it from the top. Setting it to zero will cause the element to hug the top since the specified offset distance is zero.

bottom: 0px;

The bottom offset moves an element up. Because it is offsetting it from the bottom. The greater the number the higher it will position it. Setting it to zero will cause the element to hug the bottom since the specified distance is zero.

left: 15px;

The left offset moves an element towards the right. Because it is offsetting it from the left. The greater the number the further right it pushes. Setting it to zero will cause the element to hug the left since the specified distance is zero.

right: 0px;

The right offset moves an element towards the left. Because it is offsetting it from the right. The greater the number the further left it pushes. Setting it to zero will cause the element to hug the right because the specified distance is zero.
